# Astro Bento Portfolio 🌌

A highly creative, animation-heavy single-page developer portfolio and workspace dashboard. This project features a high-performance background particle engine driven by native HTML5 Canvas vector mathematics that responds dynamically to cursor velocity. 

Clicking "Explore Workspace" dissolves the landing layer, shatters the background environment, and coordinates a smooth, staggered entry sequence into a glassmorphic Bento Grid showcasing active production repositories.

## 🛠️ Tech Stack & Architecture
* **Frontend Core:** Semantic HTML5, Custom CSS Grid Layouts, Modern Vanilla JavaScript (ES6 Modules).
* **Animation Engine:** Native Canvas API (`requestAnimationFrame` loops), Anime.js Engine (Timeline sequences, staggered arrays, elastic spring physics).
* **Build Pipeline:** Node.js, Vite Bundle Manager (Optimized static deployment configuration).

## 📂 Featured Projects
1. **Support Systems Bot:** Hybrid Node.js & Python automation engine integrated with generative AI.
2. **Gladiator's Arena:** Webpage-based game implemented using HTML5, CSS3, Node.js, and Python (Flask).
3. **DevPulse System Monitor:** Live interface tracking system CPU, RAM, GPU, and Memory metrics.
